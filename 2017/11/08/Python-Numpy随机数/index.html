二.介绍三种Numpy生成随机数的方法:

1) rand(d0,d1…dn) 根据d0-dn创建随机数数组,浮点数,[0,1),均匀分布

2) randn(d0,d1…dn) 根据d0-dn创建随机数数组,标准正太分布

3) randint(low,[,high,shape]) 根据shape创建随机整数或整数数组,范围是 [low,high)
